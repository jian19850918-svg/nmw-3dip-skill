# NMW-3Dip

## 用途

根据参考图和用户提供的简短需求，创建一套原创盲盒三维 IP 角色设计方案。参考图只能用于提取设计方法，例如展示顺序、材质语言、光影语言、排版节奏、姿势张力、服装系统、表情系统和场景氛围。

不要复制现有 IP、品牌名、Logo、角色轮廓、五官比例、服装组合，或参考图中具有明显识别度的标志性元素。

## 必要输入

允许用户输入不完整，并可根据参考图补全缺失的设计细节。但是，如果用户明确说主体和调性会由他提供，就不要自行发明核心 IP 主体或调性。

最低有效输入：

1. 参考图片或图片路径。
2. 用户给出的 IP 主体，例如原创玩具女孩、云朵生物、猫形角色、小怪物、机器人、精灵或其他角色方向。
3. 用户给出的调性，例如柔软、奇怪、可爱、潮流、治愈、叛逆、神秘、高级、运动、复古或未来感。

可选输入：

- 品牌名或项目名。
- 场景关键词。
- 必须包含的道具、服装、头饰、动作或材质。
- 必须避免的内容。
- 用户是只要提示词，还是要直接生成图片。

除非用户主动提供目标人群，否则忽略目标人群。默认不要询问目标人群。

## 版权与安全规则

始终应用以下限制：

- 创建原创 IP，不要做成任何参考角色的衍生版。
- 不要使用真实品牌 Logo、运动队标志、奢侈品牌名或现有角色名。
- 不要复刻参考图中的精确轮廓、五官布局、服装套装、头饰组合、配色分区、海报文案或产品包装。
- 只能把参考图描述为方法灵感，例如多套换装阵列、四视图设定板、表情贴纸、材质渲染或电影感光影。
- 如果用户要求过度模仿某个知名 IP 或品牌，要转向更安全的原创方向，只保留宽泛气质。
- 生成图片中的文字应尽量少且通用，除非用户提供原创品牌文字。优先使用占位文字，例如 ORIGINAL IP、SERIES 01、CHARACTER FILE，或用户自己的项目名。
- 避免水印、复制签名、社交媒体账号、第三方 Logo 和可读的虚假品牌标志。

## 参考图分析流程

当用户提供图片时，检查图片并只提取以下可复用设计信号：

1. 形体语言：圆润程度、头身比、四肢简化方式、眼睛形状、嘴部风格、轮廓平衡。
2. 材质语言：搪胶玩具光泽、软胶、树脂、植绒、透明塑料、织物针织、金属配件、哑光涂装。
3. 光影语言：棚拍柔光、轮廓光、电影感冷暖对比、产品展示阴影、夜景发光、高调白底目录光。
4. 排版语言：主视觉渲染加文字、四视图、贴纸表情板、阵列网格、细节标注、过程线稿、场景海报、系列目录。
5. 延展逻辑：换装、头饰、道具、动作、陪伴小角色、盲盒变体、隐藏款。
6. 氛围：活泼、高级、治愈、充满能量、电影感、收藏感、适合社交媒体传播。

在输出 8 张设计板之前，先总结参考图分析。

## 默认输出

除非用户另有要求，默认输出 8 张 16:9 设计板。每张设计板必须包含：

- 设计板名称。
- 设计目的。
- 排版说明。
- 视觉内容。
- 材质与光影方向。
- 中文提示词。
- 英文提示词。
- 负面提示词。

固定使用以下设计板顺序：

1. IP 世界观主视觉。
2. 单体角色档案页。
3. 平面与三维四视图设定板。
4. 表情与动作贴纸板。
5. 盲盒换装变体阵列。
6. 材质与配件细节板。
7. 运营海报主视觉。
8. 电影感三维场景应用。

## 设计板要求

### 1. IP 世界观主视觉

展示原创 IP 的世界、核心情绪、主角、陪伴物和品牌级第一印象。使用有冲击力的 16:9 构图，适合提案文档和发布页。

### 2. 单体角色档案页

展示一个放大的三维角色渲染图，并包含原创名称、性格标签、基础档案、标志性道具和干净的图文排版。既要体现角色吸引力，也要解释设计重点。

### 3. 平面与三维四视图设定板

展示正面、半侧面、侧面和背面。包含一排平面卡通图和一排三维渲染图。比例必须一致，方便后续建模。

### 4. 表情与动作贴纸板

展示适合社交运营的平面卡通表情和小动作：打招呼、开心、思考、生气、犯困、哭泣、兴奋、再见，以及一个贴合用户 IP 设定的专属动作。

### 5. 盲盒换装变体阵列

展示 8 到 12 个原创变体，每个变体有不同服装、头饰、道具和色彩点缀。包含一个隐藏款。每个变体都要有收藏感，同时保持同一个基础轮廓。

### 6. 材质与配件细节板

展示眼睛、脸部涂装、头饰、服装纹理、手持道具、鞋子、底座和包装材质的局部特写。根据需要强调搪胶光泽、软胶、树脂、织物、植绒、透明件或金属件。

### 7. 运营海报主视觉

为新品发布、上新、节日、联名或社交活动创建一张活泼的宣传海报。使用有动作感的姿势、清晰的大标题区域、强烈但干净的色彩节奏，以及明确的前景、中景、背景层次。

### 8. 电影感三维场景应用

把 IP 放入一个电影感场景中，强调氛围、景深、阴影、轮廓光、环境叙事和高级收藏品质感。

## 提示词风格

中文提示词应具体、可执行，能直接用于生产。英文提示词应保留适合图像模型识别的核心视觉词。

每条提示词都应说明：

- 16:9 横版构图。
- 用户提供的原创 IP 主体和调性。
- 根据设计板类型选择三维盲盒收藏品模式，或平面卡通模式。
- 排版结构。
- 材质。
- 光影。
- 色彩方案。
- 镜头角度。
- 背景或展示环境。
- 不复制现有 IP、不使用真实 Logo、不要水印。

## 如果用户要求直接生成图片

除非用户明确说“立即生成”，否则先输出 8 张设计板方案并请用户确认。如果直接生成图片，应一次生成一张或小批量生成，让用户能在全部生成完之前修正方向。

## 最终回复格式

使用以下结构：

1. 参考图分析摘要。
2. 根据用户主体和调性推导出的原创 IP 方向。
3. 8 张设计板方案表。
4. 每张设计板的完整提示词。
5. 版权安全说明，以及明确说明哪些内容是故意没有复制的。
6. 下一步：确认提示词，或生成选定的设计板。
