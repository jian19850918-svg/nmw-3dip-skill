# NMW-3Dip Skill

NMW-3Dip 是一个 Codex Skill，用于根据参考图和简短 IP 需求，生成原创盲盒三维 IP 角色全案。

它适合以下任务：

- 盲盒 IP 角色设计
- 三维 IP 角色设计
- 吉祥物设计
- 潮玩收藏品设计
- 角色档案页
- 四视图设定板
- 表情动作贴纸板
- 运营海报主视觉
- 电影感三维场景图

## 核心原则

参考图只用于提取设计方法，例如材质、光影、排版、动作、换装、表情和展示顺序。

不要复制现有 IP、品牌名、Logo、角色轮廓、五官比例、服装组合或其他高识别度元素。

## 安装方式

把整个 `nmw-3dip` 文件夹复制到你的 Codex skills 目录中：

```text
C:\Users\<你的用户名>\.codex\skills\nmw-3dip
```

目录结构应类似：

```text
nmw-3dip/
  SKILL.md
  README.md
  LICENSE
  agents/
    openai.yaml
```

## 使用方式

在 Codex 中提供参考图，并说明你想要的 IP 主体和调性，例如：

```text
使用 $nmw-3dip，根据这几张参考图，为我生成一个原创盲盒三维 IP。
主体是一个戴锦鲤头套的小男孩，穿米色钓鱼翁蓑衣，调性是治愈、柔和、收藏级潮玩。
```

默认输出 8 张 16:9 设计板方案：

1. IP 世界观主视觉
2. 单体角色档案页
3. 平面与三维四视图设定板
4. 表情与动作贴纸板
5. 盲盒换装变体阵列
6. 材质与配件细节板
7. 运营海报主视觉
8. 电影感三维场景应用
